# print
## simple print
```python
print("Ahmad")
>>> Ahmad
```

## multi print
```python
print("Ahmad", "Ali", "Akbar")
>>> Ahmad Ali Akbar
```

## Seprator
```python
print("Ahmad", "Ali", "Akbar", sep=" - ")
>>> Ahmad - Ali - Akbar
```

## End
```python
print("Ahmad", "Ali", "Akbar", sep=" - ", end="/")
>>> Ahmad - Ali - Akbar/
```
