# string
##  Upper
```py title="upper case"
name = "Apple"
print(name.upper())
```
output:
```
APPLE
```

## Lower
```python
name = "Apple"
print(name.Lowwer())
```
output:
```
apple
```

## Title
```python
name = "in the name of god"
print(name.title())
```
output:
```
In The Name Of God
```

## Wrapping String
```python
name1 = "Ahmad"
print(name1)

name2 = 'Ahmad'
print(name2)
```
output:
```
Ahmad
Ahmad
```
