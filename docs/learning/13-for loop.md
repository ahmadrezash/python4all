## Simple For Loop
```python
names = ["Ahmad", "Mohammad", "Ali", "Hamed"]

for name in names:
	print(name)
```

```python
for name in names:
	if name.startwith("A"):
		print(name)
```


## Break
```python
for name in names:
	if name.startwith("M"):
		break
```


## Continue
```python
for name in names:
	if name.startwith("M"):
		continue
	else:
		print(name)
```

## Range
```python
for i in nums:
	print(i)
```

```python
for i in range(12):
	print(i)
```

```python
for i in range(6, 12, 2):
	print(i)
```
